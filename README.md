# Pit Boss

Pit Boss scrapes the live BattleBots Pro League season with Bright Data, publishes
pre-registered win predictions before every episode, and publicly scores itself —
a fight-analytics engine you can hold accountable.

*Built for #BattleBotsDev. Full docs, live demo, and track record coming as the season airs.*

License: source-available, **gambling use prohibited** — see [LICENSE](LICENSE).
Pit Boss is an entertainment and data-engineering project — a public test of
forecasting methodology on a pre-taped TV show. Not betting advice. Please don't
gamble with it.
